📖 LiteraRed — Red Social Literaria
Una plataforma diseñada para amantes de la literatura. Intercambia libros virtualmente, debate en foros temáticos y lleva tu biblioteca a cualquier dispositivo.

✨ Características principales
📚 Biblioteca Virtual : Comparte archivos subiéndolos a servidores externos (MediaFire, Mega, etc.) y comparte el enlace.
💬 Foros Literarios : Espacios de discusión por géneros y autores.
🔄 Sincronización Multiplataforma : Inicia sesión en cualquier dispositivo y tus datos (perfil, posts, notas) se sincronizan en la nube.
📱 PWA Instalable : Funciona como aplicación nativa en móviles y escritorio.
🌙 Modo Oscuro : Paleta cálida basada en tonos sepia, cuero, crema y esmeralda.
✍️ Notas de Lectura : Crea y exporta tus notas en formato .txt.
🛠️ Stack Tecnológico
Interfaz de usuario : HTML5, Tailwind CSS, JavaScript (SPA estándar)
Backend : Funciones sin servidor de Vercel (Node.js)
Base de datos : Vercel KV (Redis)
Almacenamiento de Archivos : Servidores externos (La aplicación no almacena archivos)
Autenticación : JWT simple + Bcrypt
